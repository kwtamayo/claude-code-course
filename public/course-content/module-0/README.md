**Estimated Time:** 30 minutes  
**Prerequisites:** None - this is the starting point!

## Overview

Welcome to the Claude Code Course! This module ensures everyone starts from the same baseline by:

1. Scanning your current development environment
2. Identifying potential conflicts
3. Safely removing conflicting tools
4. Installing exactly what you need for this course

## Important Notes

### Safety First
- The cleanup script creates backups of everything it removes
- You can restore your previous setup if needed
- Nothing is deleted permanently without your confirmation

### What We'll Remove (If Present)
- Old versions of Node.js
- Conflicting Python installations
- Outdated Git configurations
- Previous IDE settings that might interfere

### What We Won't Touch
- Your personal files and documents
- Production work environments
- Unrelated applications
- System-critical software
